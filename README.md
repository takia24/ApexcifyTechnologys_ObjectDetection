# 🚀 Real-Time Object Detection, Tracking & Counting System (YOLOv8)

## 📌 Project Overview

This project is a real-time computer vision system developed using **YOLOv8** and **OpenCV**.
It detects, tracks, and counts objects using a webcam feed, along with an alert system and performance monitoring.

---

## 🎯 Features

* 🔍 Real-time object detection using YOLOv8
* 🎯 Object tracking with unique IDs
* 🔢 Line-crossing based object counting
* 🔔 Alert system on object detection
* ⚡ FPS (Frames Per Second) display
* 🎥 Automatic video recording of output

---

## 📷 Output

* Detects objects like person, phone, etc.
* Tracks objects with unique IDs
* Counts objects when crossing a defined line
* Displays FPS and saves output video

---

## 📊 Learning Outcomes

* Object detection using deep learning
* Real-time video processing
* Object tracking algorithms
* Performance optimization (FPS)
* Building practical AI applications
